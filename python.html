Variable

যে সকল জিনিসের মান পরিবর্তন হতে পারে তাকে Variable বলে। যেমনঃ Facebook এর Likes সংখ্যা, দেশর জনসংখ্যা ইত্যাদি।

Python এ ৩ ধরনের Variable আছে। যথাঃ

  1. String Type Variable
  2. Number Type Variable
  3. Boolean Type Variable

String Type Variable:

যে সকল variable এ কোনো Text থাকে তাকে String Type Variable বলে। যেমনঃ name = "Mohibbullah"

String Type Variable লেখার জন্য single quotes ('') বা double quotes ("") অথবা triple quotes ("' '") ব্যবহার করা হয়।

তবে সব সময় double quotes ব্যবহার করা ভালো।

Number Type Variable:

যে সকল এ variable কোনো সংখ্যা থাকে তাকে Number Type Variable বলে। যেমনঃ age = 21

Number Type Variable ২ প্রকার। যথাঃ

  1. Integer Type: যে সকল সংখ্যায় কোনো ভগ্নাংশ থাকে না। যেমনঃ age = 21
  2. Floating Type: যে সকল সংখ্যায় ভগ্নাংশ থাকে । যেমনঃ height = 5.5

Boolean Type Variable:

যে সকল variable এর value শুধুমাত্র True অথবা False হতে পারে তাকে Boolean Type Variable বলে। যেমনঃ isHot = True

User Input & Output

User এর কাজ থেকে input নেওয়ার জন্য input function ব্যবহার করতে হয়। এবং output দেখানোর জন্য print function ব্যবহার করতে হয়।

Notes: ১। Python এ input সব সময় string আকারে গ্রহন করে। তাই অন্য কোনো formate এ input গ্রহন করতে হলে তাকে keyword ব্যবহার করে convert করে নিতে হবে। যেমনঃ int => Integer এ convert করার জন্য। ২। input গ্রহনের সময় কিছু extra information দেওয়া ভালো যাতে user বুঝতে পারে।
